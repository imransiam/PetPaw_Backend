# PawMart Backend

This repository contains the backend for the PawMart web application.  
It provides API endpoints that are used by the frontend.

Live API:  
https://assignment10-backend-three.vercel.app/

---

## About the backend

The backend is built using Node.js and Express.  
It follows a simple REST structure and is kept lightweight on purpose.
